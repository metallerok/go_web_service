### Установка зависимостей
```
make install
```


### Миграции SQL базы данных
Необходимо установить средство управления миграциями SQL базы данных: [atlasgo.io](https://atlasgo.io/)

#### Создание файла миграции
```
make migration
```

#### Обновление схемы базы данных
```
make migrate
```

### Запуск Web-сервиса
```
make run_web
```
